第一部分：代數變換與純量比例

1. 若 $5x - 2y = 3x + 4y$，且 $xy \neq 0$，求 $(x+y) : (x-y)$ 之比值。
2. 設 $2x : 3y : 4z = 3 : 4 : 5$，求 $(x+2y) : z$ 的最簡整數比。
3. 已知 $x, y$ 為正數，滿足 $x : y = 2 : 1$ 且 $2x + y = 20$，求 $(x+1) : (y+1)$ 之比值。
4. 若 $\frac{a}{2} = \frac{b}{3} = \frac{c}{4}$ 且 $a^2 + b^2 + c^2 = 116$，求 $a + b + c$ 之所有可能值。
5. 設 $x : y = \frac{1}{2} : \frac{1}{3}$，$y : z = 0.4 : 0.6$，求連比 $x : y : z$。
6. 已知 $(x+y) : (y+z) : (z+x) = 5 : 7 : 8$，求 $x : y : z$ 之比。
7. 若三角形三邊長 $a, b, c$ 滿足 $3a = 4b = 6c$，且周長為 $54$，求最長邊之長。
8. 設 $ab \neq 0$，若 $\frac{3a-2b}{a+4b} = \frac{2}{3}$，求 $a^2 : b^2$。
9. 若 $\frac{M}{45} = \frac{3}{5}$ 且 $\frac{60}{N} = \frac{3}{5}$，求 $M + N$ 之值。
10. 已知 $x, y, z$ 均不為 $0$，且 $\frac{1}{x} : \frac{1}{y} : \frac{1}{z} = 2 : 3 : 4$。若 $x + y + z = 156$，求 $z$ 之值。
11. 設 $x : y = 3 : 4$，$y : z = 2 : 5$。若 $x + y + z = 102$，求 $x \cdot z$ 之值。
12. 若 $x, y, z$ 為正數，且 $x^2 : y^2 : z^2 = 1 : 4 : 9$，求 $\frac{x+y}{z}$ 之值。
13. 若 $(2x+y) : (x-y) = 5 : 1$，求 $x : y$ 之比。
14. 設 $a : b : c = 1 : 2 : 3$，求 $\frac{ab+bc+ca}{a^2+b^2+c^2}$ 之比值。
15. 已知 $2a = 3b$ 且 $4b = 5c$，求 $a : c$ 之比值。

第三部分：實務應用與變動比例邏輯

31. 檸檬水配方：水是糖的 $4$ 倍，糖是檸檬汁的 $2$ 倍。若用 $3$ 杯檸檬汁，需多少杯水？
32. 瑪麗亞收藏紅、綠、藍大理石。紅是綠的一半，藍是綠的兩倍。若總數介於 $20 \sim 30$，總數為何？
33. 某競賽男女比原為 $5 : 4$。男生加 $2$ 人、女生減 $2$ 人後比變為 $3 : 2$，求原總人數。
34. 在 $12 \times 16$ 呎房間鋪磚，邊緣用 $1 \times 1$ 呎磚，內部用 $2 \times 2$ 呎磚，求兩者數量比。
35. 行駛高速公路 $50$ 哩與一般路 $10$ 哩。高速車速為一般路 $3$ 倍。若一般路開 $30$ 分，求總時程。
36. 鹽水比例 $1 : 19$。蒸發部分水後比例變 $1 : 9$。若鹽重 $10$ 克不變，求蒸發水量。
37. 青蛙群綠黃比 $3 : 1$。若 $3$ 隻綠變黃、$5$ 隻黃變綠後比為 $4 : 1$，求原兩色數量差。
38. 甲乙丙原得 $15, 20, 40$ 元。若欲三人平分總酬勞，拿 $40$ 元者需給出多少元？
39. 池中草魚與鯉魚比 $6 : 1$。抽樣 $180$ 條中有 $30$ 條鯉魚。依此比例推估全池總魚數。
40. 產品售價 $100$ 元，利潤與成本比 $1 : 4$。若成本升 $10\%$ 且售價不變，求新利潤。
41. 水壺裝 $3/4$ 滿果汁，平分倒入 $5$ 個杯子，每杯果汁占水壺總容量之百分比為何？
42. 貓罐頭直徑增加一倍且高度減半。若每日餵食體積固定，其可餵食天數之比如何變動？
43. 沙灘上戴帽者中 $2/5$ 戴墨鏡。若戴墨鏡與戴帽者之比例已知，求墨鏡族中戴帽之比例。
44. 時鐘每小時快 $2$ 分鐘。早上 $8:00$ 對準，當該鐘顯示下午 $7:00$ 時，標準時間為何？
45. 萊拉五次測試前四次同分，第五次較高，平均 $82$ 分。求第五次分數可能之值有多少種？
46. 計算 $1, 2, 4$ 三個數字的諧和平均數（Harmonic Mean）。
47. 路徑 $15$ 哩車速 $30\text{mph}$。欲使全程（含後續 $30$ 哩）平均速率達 $50\text{mph}$，後段車速需為何？
48. 皮餅被哈洛吃掉 $1/4$，剩下的被麋鹿吃掉 $1/3$，再剩的被豪豬吃 $1/2$，求最後剩餘分率。
49. 三商品單價比 $2 : 3 : 5$，購買數量比 $5 : 4 : 2$，求最高單價商品占總花費之比例。
50. 求連乘積 $(1+\frac{1}{1})(1+\frac{1}{2})(1+\frac{1}{3})\dots(1+\frac{1}{6})$ 之最終結果。
