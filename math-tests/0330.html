第一部分：象限與點的座標

1. 已知點 $A(ab, a+b)$ 位於第二象限，且 $|a| > |b|$，請問點 $B(a, b)$ 必定落在第幾象限？

答：____________________ 第四象限

2. 若 $ab < 0$ 且 $a - b > 0$，請問點 $P(a^2b, \frac{a}{b})$ 位於第幾象限？

答：____________________ 第三象限

3. 點 $P(2k-8, -k+3)$ 到 $x$ 軸的距離是到 $y$ 軸距離的兩倍，求 $k$ 的所有可能值。

答：____________________ $\frac{19}{5}$ 或 $\frac{13}{3}$

4. 若點 $A(m, n)$ 位於第三象限，且到 $x$ 軸距離為 $4$，到 $y$ 軸距離為 $5$。求 $m$ 與 $n$ 的值。

答：____________________ $m = -5, n = -4$

5. 若 $ab < 0$ 且 $bc > 0$，請問直線 $ax + by + c = 0$ 的圖形必不通過第幾象限？

答：____________________ 第二象限

6. 請在空白處畫出方程式 $y = |x| + 1$ 的圖形。

答：____________________ 略 (V字型圖形，頂點為 $(0, 1)$，開口向上)

第二部分：平移、對稱與距離

7. 將點 $A(a, b)$ 對 $x$ 軸作對稱後，向左平移 $3$ 單位，再向上平移 $4$ 單位，最後對 $y$ 軸作對稱，得到點 $B(5, -2)$。求 $a+b$ 的值。

答：____________________ $4$

8. 點 $P(3x-1, -y+2)$ 對於直線 $x=2$ 的對稱點為 $Q(5, 3y-6)$。求 $x+y$ 的值。

答：____________________ $2$

9. 座標平面上有一點 $P(a, b)$，先向右平移 $3$ 單位，再向下平移 $5$ 單位後，剛好與原點 $O(0,0)$ 重合。求 $a, b$ 的值。

答：____________________ $a = -3, b = 5$

10. 在座標平面上，若將線段 $AB$ 向右平移 $2$ 單位、向下平移 $3$ 單位後，其兩個端點的新座標為 $A'(3, -1)$、$B'(-1, 5)$。求原線段 $AB$ 的中點座標。

答：____________________ $(-1, 5)$

11. 直線 $L$ 的方程式為 $3x - 2y = 6$。若將直線 $L$ 對 $x$ 軸作對稱，得到新直線 $L'$，請寫出 $L'$ 的方程式。

答：____________________ $3x + 2y = 6$

12. 點 $P(x, y)$ 在第四象限，且 $|x| + |y| = 6$。若 $P$ 到 $x$ 軸的距離是到 $y$ 軸距離的兩倍，求 $P$ 點座標。

答：____________________ $(2, -4)$

第三部分：圖形與面積

13. 兩直線 $L_1: x - 2y = -4$ 與 $L_2: 3x + y = 9$ 相交於點 $A$。若 $L_1, L_2$ 分別與 $x$ 軸交於 $B, C$ 兩點，求 $\triangle ABC$ 的面積。

答：____________________ $\frac{21}{2}$ 平方單位

14. 座標平面上有四條直線：$y = 2x + 4$、$y = 2x - 6$、$x = -1$、$x = 3$。請計算這四條直線所圍成的平行四邊形面積。

答：____________________ $40$ 平方單位

15. 若直線 $y = ax + b$ 通過點 $(-2, 6)$ 與 $(4, -3)$，求此直線與兩座標軸所圍成的三角形面積。

答：____________________ $3$ 平方單位

16. 直線 $L: \frac{x}{k} + \frac{y}{4} = 1$ 與兩座標軸圍成的三角形面積為 $10$。求 $k$ 的所有可能值。

答：____________________ $5$ 或 $-5$

17. 已知直線 $L: 3x - 4y = 12$。若有一平行於 $L$ 的直線 $M$ 與兩座標軸圍成的三角形面積為 $24$，求直線 $M$ 的方程式。

答：____________________ $3x - 4y = 24$ 或 $3x - 4y = -24$

第四部分：兩直線的交點與聯立方程式

18. 三條直線 $L_1: 2x - y = 5$、$L_2: x + 2y = 0$、$L_3: 3x + ay = 4$ 交於同一點，求 $a$ 的值。

答：____________________ $2$

19. 若聯立方程式 $\begin{cases} 2x + 3y = 7 \\ ax - 6y = b \end{cases}$ 有無限多組解，求 $a+b$ 的值。

答：____________________ $-18$

20. 若直線 $L_1: ax + by = 5$ 與 $L_2: 2x - 3y = c$ 互相重合，且直線 $L_1$ 通過點 $(1, -1)$，求 $a+b+c$ 的值。

答：____________________ $4$

21. 點 $A(-2, 3)$、$B(4, 7)$，若點 $P(x, 5)$ 在直線 $AB$ 上，求 $x$ 的值。

答：____________________ $1$

22. 兩直線 $L_1: y = mx + 3$ 與 $L_2: y = -2x + n$ 的交點在 $x$ 軸上，且 $L_1$ 通過點 $(2, 7)$。求 $m+n$ 的值。

答：____________________ $-1$

23. 已知聯立方程式 $\begin{cases} |x - y| = 2 \\ 2x + y = 7 \end{cases}$。請解出所有的交點座標 $(x,y)$。

答：____________________ $(3, 1)$ 或 $(\frac{5}{3}, \frac{11}{3})$

第五部分：應用問題

24. 某計程車計費方式為：起跳 $1.5$ 公里 $100$ 元，超過 $1.5$ 公里後，每 $0.5$ 公里加收 $10$ 元（不足 $0.5$ 公里以 $0.5$ 公里計算）。若小明搭乘了 $x$ 公里（$x > 1.5$ 且為 $0.5$ 的整數倍），總車資為 $y$ 元。寫出 $x, y$ 的二元一次方程式。

答：____________________ $y = 20x + 70$

25. 甲、乙兩人分別從 $A, B$ 兩地同時相向而行。設出發 $x$ 分鐘後，兩人相距 $y$ 公尺。已知方程式為 $y = -120x + 3600$。請問甲、乙兩人的速率和為每分鐘多少公尺？$A, B$ 兩地相距多遠？

答：____________________ 速率和 $120$ 公尺/分，距離 $3600$ 公尺

26. 長度為 $30$ 公分的香，點燃後每分鐘燒掉 $1.5$ 公分；長度為 $24$ 公分的香，點燃後每分鐘燒掉 $0.5$ 公分。若兩支香同時點燃，且 $x$ 分鐘後，短香的長度恰好是長香長度的 $2$ 倍，求 $x$。

答：____________________ $14.4$ (或 $\frac{72}{5}$)

27. 水箱有甲、乙兩根注水管。若單獨開甲管，水箱的水量 $y$ (公升) 與時間 $x$ (分鐘) 關係為 $y = 15x + 50$；若單獨開乙管，關係為 $y = 25x + 50$。今兩管同時開啟，幾分鐘後水箱的水量會達到 $210$ 公升？

答：____________________ $4$ 分鐘

28. 某服飾店進行促銷，全館服飾原價打七折。若成為會員，可以再折抵 $100$ 元。設原價為 $x$ 元，結帳金額為 $y$ 元。已知小美用會員身分買了一件衣服，結帳金額為 $460$ 元，求此衣服的原價。

答：____________________ $800$ 元

29. 甲、乙兩車在高速公路上同向行駛。甲車在乙車前方 $15$ 公里處，甲車的速率為 $80$ 公里/小時，乙車的速率為 $100$ 公里/小時。若兩車同時出發，求乙車需要幾分鐘才能追上甲車？

答：____________________ $45$ 分鐘

第六部分：代數變換與純量比例

30. 若 $5x - 2y = 3x + 4y$，且 $xy \neq 0$，求 $(x+y) : (x-y)$ 之比值。

答：____________________ $2$

31. 設 $x : y = 3 : 4$，$y : z = 2 : 5$。若 $x + y + z = 102$，求 $x \cdot z$ 之值。

答：____________________ $1080$

32. 已知 $x, y$ 為正數，滿足 $x : y = 2 : 1$ 且 $2x + y = 20$，求 $(x+1) : (y+1)$ 之比值。

答：____________________ $\frac{9}{5}$

33. 若 $x, y, z$ 為正數，且 $x^2 : y^2 : z^2 = 1 : 4 : 9$，求 $\frac{x+y}{z}$ 之值。

答：____________________ $1$

34. 已知 $(x+y) : (y+z) : (z+x) = 5 : 7 : 8$，求 $x : y : z$ 之比。

答：____________________ $3 : 2 : 5$

35. 設 $x : y = \frac{1}{2} : \frac{1}{3}$，$y : z = 0.4 : 0.6$，求連比 $x : y : z$。

答：____________________ $3 : 2 : 3$

36. 已知 $x, y, z$ 均不為 $0$，且 $\frac{1}{x} : \frac{1}{y} : \frac{1}{z} = 2 : 3 : 4$。若 $x + y + z = 156$，求 $z$ 之值。

答：____________________ $36$

第七部分：跨單元綜合進階題 (座標、方程式與比例)

37. 若點 $P(x, y)$ 位於第二象限，其到 $x$ 軸與 $y$ 軸的距離比為 $3 : 2$。已知 $P$ 點的座標也滿足方程式 $2x - 3y = -26$，求 $P$ 點座標。

答：____________________ $(-4, 6)$

38. 兩直線 $ax + by = 5$ 與 $ax - by = 1$ 的交點座標為 $(4, 3)$，求 $a : b$ 的最簡整數比。

答：____________________ $9 : 8$

39. 直線 $L: \frac{x}{a} + \frac{y}{b} = 1$（其中 $a>0, b>0$）與兩座標軸圍成的三角形面積為 $24$。若 $a : b = 3 : 4$，求直線 $L$ 的二元一次方程式。

答：____________________ $4x + 3y = 24$

40. 設 $x, y, z$ 皆不為 $0$，且 $2x = 3y = 4z$。若 $x - 2y + 3z = 35$，求 $x + y + z$ 的值。

答：____________________ $65$

41. 座標平面上兩點 $A(-2, 5)$ 與 $B(7, -1)$，若點 $P(x, y)$ 在線段 $AB$ 上，且 $P$ 點到 $A$ 點的水平距離與到 $B$ 點的水平距離比為 $1 : 2$，求 $P$ 點座標。

答：____________________ $(1, 3)$

42. 已知點 $(a+b, ab)$ 位於第二象限，且 $|a| : |b| = 3 : 1$。求 $(a+b) : (a-b)$ 的比值。

答：____________________ $2$

43. 將直線 $L: y = mx + k$ 向左平移 $3$ 單位，再向下平移 $2$ 單位後，新直線通過原點 $(0,0)$ 與點 $(2,4)$。求 $m : k$ 的最簡整數比。

答：____________________ $1 : -2$
